Crossfire: Ys 4.0.6 版本 正式发布

这款游戏使用Unity引擎构建，目前只支持Windows端，短期时间内没有支持其他端的计划

下载

Github下载

目前版本优点

  • 云端账号：使用网络保存您的游戏进度
  • 多关卡，多武器，完善您的游戏体验
  • 智能怪物。

配置

  • 最低配置

    • 1.系统：Windows 7 SP1 64-bit 或 Windows 8.1 64-bit 或 Windows 10 64-bit
    • 2.储存：512MB
    • 3.内存：2GB RAM
    • 4.DirectX 版本：11

  • 推荐配置：

    • 1.系统：Windows 10 64-bit 或 Windows 11 64-bit
    • 2.储存：1GB～2GB
    • 3.内存：8GB RAM
    • 4.DirectX 版本：11
