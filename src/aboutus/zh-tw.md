# 關於

## 網站簡介

- **建站時間**: 2025-02-22

## 技術實現
- **網域**: 本站使用兩個網域，`undz.cn` 和 `io.hb.cn`，兩個網域都透過阿里雲智能科技（杭州）有限公司註冊，並由 Cloudflare 提供 DNS 解析和 SSL 憑證。
- **憑證**: 兩個網域均持有 中國互聯網路資訊中心（CNNIC） 頒發的合法網域憑證，註冊人資訊已通過實名認證。
- **安全加密**：
  - 全站使用 HTTPS 加密存取。
  - SSL 憑證由 [Cloudflare](https://www.cloudflare.com/) 提供。
- 使用 [VuePress](https://vuepress.vuejs.org/zh/) 強力驅動
- **圖示**: 本站圖示由我自己設計，其餘向量圖示均使用 Iconify 的 Font Awesome 6 Solid，作者 Dave Gandy。
- **主題**: [vuepress-theme-hope](https://theme-hope.vuejs.press/zh/)
