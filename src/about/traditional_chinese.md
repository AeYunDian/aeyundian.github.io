## 技術實現
- **域名**: 本站使用兩個域名，`undz.cn`和`io.hb.cn`，兩個域名都通過阿里雲註冊，並由 Cloudflare 提供 DNS 解析和 SSL 證書。
- **證書**: 兩個域名均持有 中國互聯網絡信息中心（CNIC） 頒發的合法域名證書，註冊人信息已通過實名認證。
- **安全加密**：
  - 全站使用 HTTPS 加密訪問。
  - SSL 證書由 [Cloudflare](https://www.cloudflare.com/) 提供，採用 TLS 1.3 協議。
- **託管**: 博客源碼託管於 GitHub，並通過 GitHub Pages 自動部署
  - 源碼託管: [GitHub](https://github.com/AeYunDian/aeyundian.github.io/) 
  - 博客地址: https://undz.cn（由 GitHub Pages 自動部署）
- 使用 [VuePress](https://vuepress.vuejs.org/zh/) 強力驅動
- 圖標: 本站圖標由我自己設計。
- 主題: [vuepress-theme-hope](https://theme-hope.vuejs.press/zh/)
