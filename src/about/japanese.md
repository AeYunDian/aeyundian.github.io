## 技術実装
- **ドメイン**: 当サイトは `undz.cn` と `io.hb.cn` の2つのドメインを使用しています。どちらも Alibaba Cloud で登録し、Cloudflare が DNS 解決と SSL 証明書を提供しています。
- **証明書**: 両ドメインとも中国インターネットネットワーク情報センター (CNIC) 発行の有効なドメイン証明書を保有し、登録者情報は実名認証済みです。
- **セキュリティ暗号化**:
  - 全ページ HTTPS 暗号化アクセス。
  - SSL 証明書は [Cloudflare](https://www.cloudflare.com/) 提供、TLS 1.3 プロトコルを使用。
- **ホスティング**: ブログのソースコードは GitHub でホスティングされ、GitHub Pages により自動デプロイされます。
  - ソースコード: [GitHub](https://github.com/AeYunDian/aeyundian.github.io/)
  - ブログアドレス: https://undz.cn（GitHub Pages により自動デプロイ）
- [VuePress](https://vuepress.vuejs.org/zh/) を利用
- **アイコン**: 自分でデザインしました。
- **テーマ**: [vuepress-theme-hope](https://theme-hope.vuejs.press/zh/)
