# À propos

## Présentation du site
- **Nom** : undz
- **Date de création** : 2025-02-22

## Réalisation technique
- **Domaine** : Ce site utilise deux domaines, `undz.cn` et `io.hb.cn`. Les deux sont enregistrés chez Alibaba Cloud, et Cloudflare fournit la résolution DNS et les certificats SSL.
- **Certificats** : Les deux domaines possèdent des certificats valides délivrés par le Centre d'Information du Réseau Internet de Chine (CNIC), et les informations des titulaires ont été vérifiées.
- **Chiffrement sécurisé** :
  - Accès intégral en HTTPS.
  - Certificats SSL fournis par [Cloudflare](https://www.cloudflare.com/), utilisant le protocole TLS 1.3.
- **Hébergement** : Le code source du blog est hébergé sur GitHub et déployé automatiquement via GitHub Pages.
  - Code source : [GitHub](https://github.com/AeYunDian/aeyundian.github.io/)
  - Adresse du blog : https://undz.cn (déployé automatiquement par GitHub Pages)
- Propulsé par [VuePress](https://vuepress.vuejs.org/zh/)
- **Icônes** : Conçues par moi-même.
- **Thème** : [vuepress-theme-hope](https://theme-hope.vuejs.press/zh/)
