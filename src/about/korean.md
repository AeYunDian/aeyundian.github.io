## 기술 구현
- **도메인**: 본 사이트는 `undz.cn`과 `io.hb.cn` 두 개의 도메인을 사용합니다. 두 도메인 모두 Alibaba Cloud를 통해 등록되었으며, Cloudflare가 DNS 해석과 SSL 인증서를 제공합니다.
- **인증서**: 두 도메인 모두 중국 인터넷 네트워크 정보 센터(CNIC)에서 발급한 유효한 도메인 인증서를 보유하고 있으며, 등록자 정보는 실명 인증되었습니다.
- **보안 암호화**:
  - 전체 사이트 HTTPS 암호화 접속.
  - SSL 인증서는 [Cloudflare](https://www.cloudflare.com/)에서 제공하며, TLS 1.3 프로토콜을 사용합니다.
- **호스팅**: 블로그 소스 코드는 GitHub에 호스팅되며, GitHub Pages를 통해 자동 배포됩니다.
  - 소스 코드: [GitHub](https://github.com/AeYunDian/aeyundian.github.io/)
  - 블로그 주소: https://undz.cn (GitHub Pages에 의해 자동 배포됨)
- [VuePress](https://vuepress.vuejs.org/zh/) 기반
- **아이콘**: 직접 디자인했습니다.
- **테마**: [vuepress-theme-hope](https://theme-hope.vuejs.press/zh/)
