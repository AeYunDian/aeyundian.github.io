## Technical Implementation
- **Domain**: This site uses two domains, `undz.cn` and `io.hb.cn`. Both are registered through Alibaba Cloud, with Cloudflare providing DNS resolution and SSL certificates.
- **Certificates**: Both domains hold valid domain certificates issued by the China Internet Network Information Center (CNIC), and registrant information has been verified.
- **Security Encryption**:

  - SSL certificates provided by [Cloudflare](https://www.cloudflare.com/), using TLS 1.3 protocol.
- **Hosting**: The blog source code is hosted on GitHub and automatically deployed via GitHub Pages.
